# 本地编辑和编译

## 编辑

主要编辑这个文件：

```powershell
D:\Master\1_找工\简历\resume\resume-zh_CN.tex
```

可以用 VS Code 打开整个目录：

```powershell
code "D:\Master\1_找工\简历\resume"
```

## 编译中文简历

在 PowerShell 里运行：

```powershell
cd "D:\Master\1_找工\简历\resume"
.\build-zh_CN.ps1
```

生成结果：

```powershell
D:\Master\1_找工\简历\resume\resume-zh_CN.pdf
```

## 直接编译命令

也可以不用脚本，直接运行：

```powershell
cd "D:\Master\1_找工\简历\resume"
xelatex resume-zh_CN.tex
xelatex resume-zh_CN.tex
```

运行两次是为了让目录、链接、引用等辅助信息刷新完整。
