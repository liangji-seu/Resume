# Build

This repository contains the LaTeX source and local assets needed to build the resume PDF.

## Requirements

- XeLaTeX
- A TeX distribution such as MacTeX, TeX Live, or MiKTeX

## macOS/Linux

```bash
./build-zh_CN.sh
```

## Windows PowerShell

```powershell
.\build-zh_CN.ps1
```

Both scripts generate a dated PDF named:

```text
梁吉_简历_yyyy-MM-dd.pdf
```
