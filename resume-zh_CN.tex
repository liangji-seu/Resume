设计并构建高效、灵活且可扩展的深度学习模型推理框架，通过优化模型加载与推理流程，提升处理速度和资源利用效率，支撑多种业务场景下的模型推理需求。
\begin{itemize}
  \item \textbf{架构搭建：}实现支持 Llama 2/3.2、Qwen 等主流深度学习模型的推理框架，支持 YOLOv5、ResNet 等模型，支持 CPU/GPU 双后端推理与 KV-Cache 等常见优化手段。
  \item \textbf{算子调优：}针对 CUDA 算子进行深度定制和优化，实现 RMSNorm、MatMul、KV-Cache、MultiHead Attention、FlashAttention 等核心算子。
  \item \textbf{性能分析：}使用 Night-Compute、Night-System 等工具分析和调优算子性能；在 RTX 4090 上对 Qwen3-0.6B FP32 模型测试，单轮生成 332 个 decode tokens，TTFT 约 47.8ms，平均 decode latency 约 3.73ms/token，decode 吞吐约 267.9 token/s。
  \item \textbf{项目成果：}完成 Llama 2/3.2 与 Qwen 模型中多个复杂 CUDA 算子的开发和性能优化，覆盖 RMSNorm、MatMul、KV-Cache、MHA 等模块，提升推理速度与资源利用率。
  \item \textbf{框架研究：}深入学习 llama.cpp、vLLM 等大模型推理框架，对 llama.cpp 的内部实现和优化技巧进行研究，包括 PagedAttention、Continuous batching 等机制。
\end{itemize}
